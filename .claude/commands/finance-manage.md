# 🦞 OpenClaw 财务管理 — 财务健康分析与规划

分析项目/公司的财务状况，提供资金管理和预算优化建议。

## 参数

- `$ARGUMENTS` — 必填，财务信息描述，格式：`月收入|月支出|现金余额|月度预算`。
  示例：`96000|72000|125000|80000`
  或直接描述当前财务状况，如 `刚启动的SaaS，目前月收入$2000，月支出$5000，现金$30000`

## 执行步骤

### 第一阶段：财务健康诊断

1. 解析输入的财务数据
2. 计算核心指标：
   - 💵 月度净利润 = 月收入 - 月支出
   - 🔥 燃烧率 = 月支出（亏损时）
   - 📅 资金跑道 = 现金余额 / 燃烧率（月）
   - 📊 收入/支出比 = 月收入 / 月支出
   - 💹 增长率（如有历史数据）

3. 健康评估：

| 指标 | 健康 | 警告 | 危险 |
|------|------|------|------|
| 资金跑道 | > 6个月 | 3-6个月 | < 3个月 |
| 收入/支出比（增长期） | > 0.5 | 0.3-0.5 | < 0.3 |
| 收入/支出比（成熟期） | > 1.0 | 0.8-1.0 | < 0.8 |
| 毛利率 | > 60% | 40-60% | < 40% |

### 第二阶段：支出结构分析

1. 使用 WebSearch 调研同类项目的成本参考：
   - "{行业/项目类型} startup cost breakdown"
   - "{行业/项目类型} typical expenses SaaS"

2. 建议支出优化方向：
   - 基础设施成本优化
   - AI 工具/API 调用成本控制
   - 人力成本效率提升
   - 订阅服务精简

### 第三阶段：预算管理建议

根据自主支出权限矩阵提供管理建议：

```
支出权限建议：
  < ¥700 / $100       → 团队自主决策，记录日志
  ¥700-7,000 / $100-1K → 执行后 24h 内汇报
  ¥7K-70K / $1K-10K   → 需审批后执行
  > ¥70K / $10K       → 需双重审批

特殊规则：
  - 同一供应商 7 天累计 > ¥5,000 / $700 → 升级审批
  - 月支出超预算 110% → 冻结非必要支出
```

### 第四阶段：现金流预测

生成 6 个月现金流预测（表格形式）：

| 月份 | 期初余额 | 预计收入 | 预计支出 | 净现金流 | 期末余额 |
|------|---------|---------|---------|---------|---------|

假设条件需明确列出（保守/中性/乐观三种情景）。

### 第五阶段：行动建议

根据分析结果输出具体行动清单：

```
立即行动（本周）：
  - [ ] 行动1
  - [ ] 行动2

短期优化（本月）：
  - [ ] 行动1
  - [ ] 行动2

长期规划（本季度）：
  - [ ] 行动1
  - [ ] 行动2
```

### 输出格式

请输出完整的财务分析报告，包含以上五个阶段。使用表格和可视化文字（进度条、图标）使报告直观易读。最后给出一个综合健康评分（0.00-1.00）和一句话总结。
