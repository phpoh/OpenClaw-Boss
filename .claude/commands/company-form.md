# 🦞 OpenClaw 公司组建 — 搭建创业基础设施

为已验证的商业机会搭建公司法律、财务和运营基础设施。

## 参数

- `$ARGUMENTS` — 必填，格式为：`公司名称|实体类型|注册地|初始预算(元)`。
  示例：`智简科技|有限责任公司|上海|500000`
  如参数不完整，请交互式引导用户补充。

## 执行步骤

### 第一阶段：解析参数并合规预检

1. 解析用户提供的参数（公司名称、实体类型、注册地、初始预算）
2. 使用 WebSearch 进行合规预检：
   - 搜索 "{公司名称} 商标注册 中国商标网" 检查商标冲突
   - 搜索 "{公司名称} .com 域名注册" 检查域名可用性
   - 搜索 "{注册地} {实体类型} 注册要求 费用 流程"
   - 搜索 "{注册地} 公司注册 所需材料 时间"

3. 输出合规预检报告，包括：
   - 商标冲突检查结果
   - 域名可用性检查结果（优先 .cn / .com.cn / .com）
   - 注册要求和费用概览

### 第二阶段：注册方案制定

根据注册地推荐注册路径：

**中国大陆（优先推荐）：**
- 有限责任公司 / 个体工商户注册流程
- 所需材料清单（身份证、租赁合同、公司章程等）
- 注册费用估算（刻章、银行开户、代理费）
- 时间周期（一般 3-7 个工作日）
- 园区/孵化器注册优惠（如适用）

**自贸区/经济特区：**
- 海南自贸港（税收优惠）
- 深圳前海（科技企业优惠）
- 上海临港（人才政策）
- 各地园区返税政策对比

**海外注册（如有需要）：**
- 香港（费用约 3000-8000 港币）
- 新加坡（费用约 2000-5000 新币）
- 美国 Delaware（费用约 $500 + 州费）

### 第三阶段：品牌资产方案

1. 使用 WebSearch 搜索同行业品牌风格参考：
   - "{行业} 品牌设计 logo 参考"
   - "{行业} 品牌配色 视觉风格"

2. 生成品牌建议：
   - 3 个 Logo 设计方向（描述概念，不含实际图片）
   - 推荐配色方案（含 HEX 色值）
   - 推荐字体搭配（中文优先：思源黑体、阿里巴巴普惠体等）
   - 品牌调性关键词

### 第四阶段：技术基础设施清单

输出完整的启动清单：

```
法律与合规：
  [ ] 公司注册（工商局/市场监管局）
  [ ] 税务登记（金税盘、发票申领）
  [ ] 银行开户（对公账户）
  [ ] 社保公积金开户
  [ ] ICP备案（如需网站）
  [ ] 许可证申请（如需，如EDI、ICP许可证）

品牌与域名：
  [ ] 域名注册（优先 .com / .cn / .com.cn）
  [ ] Logo 设计
  [ ] 品牌指南

技术工具：
  [ ] 代码仓库（Gitee / GitHub）
  [ ] 项目管理（飞书 / Teambition / Tower）
  [ ] 通讯工具（飞书 / 企业微信 / 钉钉）
  [ ] 监控告警（阿里云ARMS / 腾讯云监控）
  [ ] 文档协作（飞书文档 / 语雀 / 腾讯文档）

收款与财务：
  [ ] 支付接入（支付宝 / 微信支付）
  [ ] 记账软件配置（用友 / 金蝶 / 柠檬云）
  [ ] 发票管理（电子发票）
  [ ] 预算告警设置
```

### 第五阶段：预算分配

基于初始预算（人民币）生成建议分配方案（表格形式）：
- 法律注册费用（工商注册、刻章、代理）
- 域名与品牌
- 技术工具订阅
- 云服务费用（阿里云/腾讯云/华为云）
- 预留运营资金
- 应急储备

### 输出格式

请输出完整的新公司启动方案，包含以上五个阶段的结果。每个阶段用 Markdown 标题分隔。最后附上一个时间线（甘特图式文字描述），标注哪些步骤需要人类线下操作。
