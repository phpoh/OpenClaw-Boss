# 🦞 OpenClaw 公司组建 — 搭建创业基础设施

为已验证的商业机会搭建公司法律、财务和运营基础设施。

## 参数

- `$ARGUMENTS` — 必填，格式为：`公司名称|实体类型|注册地|初始预算(USD)`。
  示例：`智简科技|有限责任公司|CN-SH|50000`
  如参数不完整，请交互式引导用户补充。

## 执行步骤

### 第一阶段：解析参数并合规预检

1. 解析用户提供的参数（公司名称、实体类型、注册地、初始预算）
2. 使用 WebSearch 进行合规预检：
   - 搜索 "{公司名称} 商标注册" 检查商标冲突
   - 搜索 "{公司名称} .com 域名" 检查域名可用性
   - 搜索 "{注册地} {实体类型} 注册要求 费用 流程"

3. 输出合规预检报告，包括：
   - 商标冲突检查结果
   - 域名可用性检查结果
   - 注册要求和费用概览

### 第二阶段：注册方案制定

根据注册地推荐注册路径：

**中国大陆（CN）：**
- 有限责任公司注册流程
- 所需材料清单
- 注册费用估算
- 时间周期

**美国（US-Delaware）：**
- Stripe Atlas / Firstbase 自动化注册方案
- 费用：$500 + 州费
- C-Corp vs LLC 选择建议

**新加坡（SG）：**
- Stripe Atlas 方案
- 费用估算：$300-$800

### 第三阶段：品牌资产方案

1. 使用 WebSearch 搜索同行业品牌风格参考：
   - "{行业} startup branding design"
   - "{行业} logo design trends"

2. 生成品牌建议：
   - 3 个 Logo 设计方向（描述概念，不含实际图片）
   - 推荐配色方案（含 HEX 色值）
   - 推荐字体搭配
   - 品牌调性关键词

### 第四阶段：技术基础设施清单

输出完整的启动清单：

```
法律与合规：
  [ ] 公司注册
  [ ] 税务登记
  [ ] 银行开户
  [ ] 许可证申请（如需）

品牌与域名：
  [ ] 域名注册
  [ ] Logo 设计
  [ ] 品牌指南

技术工具：
  [ ] 代码仓库（GitHub/GitLab）
  [ ] 项目管理（Linear/Notion）
  [ ] 通讯工具（Slack/飞书）
  [ ] 监控告警（Sentry）
  [ ] 文档协作（Notion/Confluence）

收款与财务：
  [ ] 支付接入（Stripe/支付宝/微信支付）
  [ ] 记账软件配置
  [ ] 预算告警设置
```

### 第五阶段：预算分配

基于初始预算生成建议分配方案（表格形式）：
- 法律注册费用
- 域名与品牌
- 技术工具订阅
- 预留运营资金
- 应急储备

### 输出格式

请输出完整的新公司启动方案，包含以上五个阶段的结果。每个阶段用 Markdown 标题分隔。最后附上一个时间线（甘特图式文字描述），标注哪些步骤需要人类线下操作。
